# 基于Java Web网上购物平台

## 项目简介
网上购物平台是基于Java Web技术，SSH框架，JSP技术，Mysql数据库的在线购物平台。包含前台（销售平台），后台（对商品、订单、用户进行管理）。角色上分为：普通用户、管理员两种角色。适合作为毕业设计、大作业、课程设计。

https://bysj.bsitem.com/#/home/goodDetail?goodId=1060

## 技术介绍
Java;SSH;JSP;Html/Css/JavaScript;Mysql

## 开发工具
1.IntelliJ IDEA或者Myeclipse 3.Navicat 3.Mysql 4.JDK8

## 项目资料包含
1. 前后端源代码
2. 开题报告（任务书）+ 配套论文（一万两千字以上）+ 答辩PPT
3. 远程控屏调试（向日葵或者toDesk控屏）
4. 一键启动项目（双击运行演示，无需搭建复杂环境）
5. 项目讲解视频教程
